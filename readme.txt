== Description ==

A simple plugin to help book authors and other writers produce a professional-quality Press Kit and display it with a shortcode. 

The main purpose of a Press Kit is to make it as easy as possible for members of the media to provide publicity for you or your books. 


= Live Demo =

To see it in action, check out the [Live Demo](https://cato.io/writer-press-kit-demo)


= Features =

* Projects professionalism by conforming to industry standards
* Requires no technical skills
* Fully responsive
* Will work with any theme
* Allows you to upload and include photos
* Allows you to include custom content


= How to Use =

1. Login to WordPress
2. Navigate to Tools > Writer Press Kit
3. Enter and save your content
4. Type or paste the shortcode [writer_press_kit] in a post or page (including the brackets)



For more info, vist the [Writer Press Kit home page](https://cato.io/writer-press-kit)


== Installation ==

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
